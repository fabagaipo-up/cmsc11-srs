# cmsc11-srs
A basic starting code for creating a student registration system

Features:
1. Basic starting password to unlock the menu
2. Menu consists of creating student and subject records through manual input or file import, enrolling students to subjects, updating the student and subject records, deleting, displaying and exporting them to a masterlist.csv
3. Limited to one file import and importing multiple files will lead to the removal of the previous.
4. Limited to 70 students.
5. Numbering of student records and subjects start at 001.
